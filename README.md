# Zomato Data Analysis Project

## 📌 Project Overview
This project analyzes Zomato restaurant data to understand customer preferences, restaurant trends, and rating patterns using Python.

---

## 🎯 Objective
- Analyze restaurant data from Zomato
- Identify trends in ratings, pricing, and cuisines
- Understand customer behavior

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

---

## 📊 Dataset
- Zomato Dataset (CSV file)
- Contains information about restaurants, ratings, cost, location, and cuisines

---

## 🔍 Key Analysis Performed
- Data cleaning and preprocessing
- Handling missing values
- Rating distribution analysis
- Cost vs Rating comparison
- Most popular cuisines
- City-wise restaurant analysis

---

## 📈 Key Insights
- Restaurants with moderate pricing have better ratings
- Some cuisines are more popular than others
- Ratings are influenced by location and pricing

---

## 🚀 Conclusion
This project helps understand how different factors like cost, location, and cuisine affect restaurant ratings and customer preferences.
